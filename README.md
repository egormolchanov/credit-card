# credit-card

1) Input mask for all text fields.
2) Input check.
3) Defines 3 payment systems:
                              Mastercard,
                              Visa,
                              Maestro
4) Defines 4 Russian banks:
                            Sberbank,
                            Alfa bank,
                            Tinkoff bank,
                            Vtb bank
5) Design card changes depending on the bank.
6) Adaptive layout.
